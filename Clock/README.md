# Clock Project

A minimalist analog clock web application with a grayscale color scheme.

## Files Overview

- **index.html** - Main HTML structure for the clock display
- **script.js** - JavaScript logic for clock functionality and animation
- **style.css** - Styling and layout for the clock interface

## Color Scheme

The project uses a monochrome, "liberant" grayscale theme featuring:

- **Background**: Dark gradient (#0b0b0c to #141617)
- **Rings**: Multiple grayscale gradient layers for dates, months, and weekdays
- **Hands**: Metallic grayscale gradients for hour, minute, and hands
- **Center Hub**: Light neutral color for the center dot

## Getting Started

1. Open `index.html` in your web browser
2. The clock will display the current time with animated hands

## Features

- Real-time clock display
- Smooth hand animations
- Responsive grayscale design
- Modular color reference system
